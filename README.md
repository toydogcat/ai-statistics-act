# 📊 線上智慧統計分析平台 (ai-statistics-act)

> **「專為學術論文寫作者量身打造的極簡統計神器！」**
> 結合《給論文寫作者的統計指南：傻瓜也會跑統計》核心理論與 APA 格式寫作規範，實現 100% 瀏覽器本機高速運算，免除安裝 SPSS 或編寫 R 語言代碼的痛苦。

---

## 🌟 核心功能特色

### 1. 🧠 統計決策樹導師 (Method Wizard)
* **智慧問答式引導**：依據您的研究目的、自變項（IV）數量、組別特徵（獨立 vs 相依）、依變項（DV）類型，自動推薦最合適的統計方法。
* **一鍵直達分析**：確定方法後，系統自動將您導向數據編輯器，並完成對應分析參數的變項映射設定。

### 2. 🗂️ SPSS 智慧數據編輯器 (DataGrid)
* **Excel/SPSS 風格網格**：支援線上雙擊編輯儲存格數值、重命名變項、動態增減樣本（行）與變項（列）。
* **極速數據匯入/匯出**：整合 `PapaParse`，支援拖曳或上傳您自備的 `.csv` 或 `.txt` 數據檔，並可一鍵將編輯後的結果下載回本地。
* **數據與分析雙向聯動**：一旦表格數值修改，統計運算與圖表結果將進行**即時即地（Real-time）重算渲染**。

### 3. ⚡ 100% 本機高效統計引擎 (Client-Side Stats Engine)
* **無伺服器延遲**：完全在瀏覽器本機端使用純 JavaScript 進行矩陣運算與機率分布逼近計算，保障數據隱私與運行速度。
* **支援核心分析法**：
  * **描述統計**：平均數、標準差、變異數、極值、全距、偏態、峰度。
  * **獨立樣本 t 檢定**：整合 Levene 變異數同質性檢定，自動計算未修正及 Welch 修正值，並求出 Cohen's $d$ 效果值。
  * **相依/配對樣本 t 檢定**：計算前後測差異與 Cohen's $d$ 效果值。
  * **Pearson 相關分析**：計算關聯強度 $r$、檢定值 $t$ 與雙尾 $p$ 值。
  * **單因子獨立樣本變異數分析 (One-way ANOVA)**：輸出完整的變異數來源表，並提供 **Bonferroni 事後比較 (Post-Hoc)** 計算。
  * **複線性迴歸分析 (Multiple Regression)**：支持最小平方法矩陣運算，輸出 $R^2$、調整後 $R^2$、均方與顯著性，並計算各係數的 $t$ 檢定值。
  * **調節效果分析 (Moderation)**：包含自變項與調節變項中心化（Centering）、交互乘積項預測，並實作 **單純斜率檢定 (Simple Slopes Test)**，提供高調節組（+1 SD）與低調節組（-1 SD）之顯著性計算。

### 4. 📈 即時統計交互圖表 (Chart.js Data Visualization)
* **自動圖表配對**：
  * t 檢定與 ANOVA ── 自動生成**組別平均值長條圖**。
  * 相關與迴歸 ── 自動繪製**散佈圖與線性擬合趨勢線**。
  * 調節效果 ── 自動繪製**單純斜率交互作用圖**。

### 5. ✍️ 學術級 APA 格式自動報告生成器 (APA Report Generator)
* **教科書級中文論述範本**：自動將運算結果（t 值、df、F 值、p 值、效果值）套入標準中文 APA 論文寫作語句中。
* **標準 APA 三線表**：網頁端渲染無垂直邊線之精美三線表。
* **一鍵複製**：提供「複製內文」與「複製表格」按鈕。複製表格後可**直接貼入 MS Word**，保持完美表格框線格式，不需重新排版。

## 🛠️ 技術棧與架構

* **前端核心**：React 19 + Vite 8
* **樣式系統**：Tailwind CSS v4 (採用全新 `@tailwindcss/vite` CSS-first 編譯技術)
* **圖表渲染**：Chart.js + react-chartjs-2
* **數據解析**：PapaParse
* **開源圖標**：Lucide React

## 🚀 快速啟動

### 1. 安裝依賴
在專案根目錄下運行：
```bash
npm install
```

### 2. 開發環境運行
啟動 Vite 開發伺服器：
```bash
npm run dev
```

### 3. 生產環境建構
打包優化後的靜態檔案（生成於 `dist/` 目錄，專為 GitHub Pages 部署優化）：
```bash
npm run build
```
